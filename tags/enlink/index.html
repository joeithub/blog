接口

管理员与角色页面

1.获取所有的管理员角色展示

需要一个查询所有接口
method: GET
apiurl: http://ip:port/admin/role/all

2.所有权限写在配置文件中点击新增的时候返回给前端

提供一个所有权限的接口
method: GET
apiurl: http://ip:port/admin/role/right

3.新建完成后保存到数据库

需要一个保存接口
method: POST
body: {}
apiurl: http://ip:port/admin/role/save
保存完后再返回一个查询所有

4.查看新增后此角色的权限

需要一个根据角色名查询权限的接口
method: POST
body: {“id”: “”}
apiurl: http://ip:port/admin/role/searchById

5.更新角色名或角色权限upadte

method: POST
body: {“id”: “”}
apiurl: http://ip:port/admin/role/update

6.删除角色

method: POST
body: {“id”: “”}
apiurl: http://ip:port/admin/role/delete

7.启用/禁用此角色

method: POST
